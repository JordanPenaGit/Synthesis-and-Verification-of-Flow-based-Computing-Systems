For BDD.java

If you wish to change the file it will run, at line 48 you can replace the string in "Scanner sc = new Scanner(new File("bdds/var1.bdd"));" with your desired file.
If you wish to modify the input vector for task 2.3, modify the set array on line 83.
Running "javac BDD.java && java BDD" in the bash terminal will give you all outputs asked by in the task list.


For xbar.java
Similar process, modify line 129 in the scanner to the desired file path.
input vector is at line 161.
Running "javac xbar.java && java xbar" in the bash terminal will give you all outputs asked by in the task list.
